Introducción a Git

Comando básicos

  • git init: Inicializa un repositorio
  • git status: Verifica el estado de los archivos
  • git pull: Obtiene los archivos de GitHub
  • git add: Agrega los archivos a la staging area
  • git commit: Guarda los archivos en el repostorio local
  • git push: Manda los archivos a GitHub
  • git log: Muestra el historial del commits
  • git diff: Muestra los cambios del archivo
